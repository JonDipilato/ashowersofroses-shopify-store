# A Shower of Roses Shopify Theme

Custom Shopify Online Store 2.0 theme for **A Shower of Roses** with a Cathedral Luxe visual system.

## Connect This Theme via GitHub

1. Push this repository to GitHub.
2. In Shopify Admin, open **Online Store > Themes**.
3. Select **Add theme > Connect from GitHub**.
4. Authorize GitHub (if prompted), then pick this repository and branch.
5. Click **Add theme** and open **Customize**.

## Initial Setup Checklist

1. In **Theme settings**, set colors, typography, and ornament intensity.
2. In **Header**, assign menus for:
   - Occasions menu
   - Sacraments menu
   - Utility menu
3. In the homepage sections, connect real collections/pages (no product mocks are used).
4. For About/Contact content, assign templates on Shopify pages:
   - `page.about`
   - `page.contact`
5. Publish the theme when ready.
